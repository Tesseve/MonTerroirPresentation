Mon Terroir

Connexion

Sur cette page, vous pouvez vous connecter à l'application en saisissant votre nom d'utilisateur et votre mot de passe.

Si vous n'avez pas encore vos identifiants de connexion, vous pouvez cliquer sur le bouton "Inscription" afin de procéder à l'enregistrement dans l'application.

Inscription

Sur cette page, vous pouvez vous inscrire à l'application en saisissant votre rôle (vous devez décider si vous souhaitez apparaître en tant que client ou producteur). Le rôle de producteur vous permet d'ajouter à votre profil un certain nombre de produits qui seront affichés aux clients de l'application.

Pour s'inscrire à l'application, on demande un nom d'utilisateur, un mot de passe et une photo que vous pouvez soit choisir dans la bibliothèque de votre smartphone, soit prendre sur place avec votre caméra.

Si vous disposez déjà des informations d'identification, vous pouvez faire défiler la page vers le bas pour faire apparaître le bouton de connexion qui, si vous appuyez dessus, vous mènera à la page de connexion de l'application.

Profil

Sur cette page, accessible via le menu de navigation en bas de l'écran, on peut accéder, dans le cas d'un producteur, à la sélection de ses produits et aux commentaires que les clients ont fait à ce sujet. Il y a également un bouton en bas qui permet de se déconnecter de l'application.

Produit

Sur cette page, il est possible d'afficher des informations sur un produit particulier. Vous pouvez donc voir l'image, le nom et le coût du produit.

Si le produit est affiché par le producteur qui en est propriétaire, il est possible de cliquer sur les boutons permettant de modifier le produit et de le supprimer.

Sur cette page, vous pouvez également contacter le producteur qui possède le produit affiché.

Liste de mes produits

Sur cette page, vous pouvez voir tous vos produits dans le cas où vous êtes un producteur. Chaque produit a une image, un nom, un coût et une notion de disponibilité.

Pour chaque produit, vous pouvez cliquer sur le cœur pour l'ajouter à vos produits favoris (cette fonction n'est pas encore opérationnelle pour le moment) et ainsi que sur l'icône de chat pour écrire un message directement au producteur du produit.

Création d'un produit

Sur cette page, vous pouvez créer un produit (cette fonction n'est disponible que si vous avez choisi d'être un producteur lors de votre inscription à l'application. Vous pouvez y saisir un nom, un prix, une description et ajouter les catégories auxquelles ce produit appartient. Vous pouvez également ajouter une image. Enfin, vous pouvez enregistrer le produit, qui apparaîtra alors dans les recherches des clients.

Choix catégories d'un produit

Sur cette page, vous pouvez voir le menu qui apparaît lorsque vous cliquez sur "Catégories". Comme vous pouvez le voir sur l'image, vous pouvez choisir parmi une large sélection de catégories, afin d'être aussi pertinent que possible et de répondre au mieux aux besoins de vos clients.

Page d'accueil

Il s'agit de la page d'accueil de l'application. Sur cette page, vous pouvez afficher un court message d'accueil en haut de l'écran, suivi de la date du jour. Dans l'espace libre, vous pouvez ajouter un filtre de lieu, qui vous permet de rechercher des produits en fonction du lieu choisi. Sur le reste de la page, vous pouvez voir certains produits et producteurs séparés en différentes catégories.

Liste des produits - explore

Sur cette page, une liste de produits est affichée et filtrée à partir de la recherche d'emplacement et des catégories prédéfinies de l'application.

Filtre pour liste des produits

Sur cette page, un exemple de l'utilisation des filtres.

Carte des producteurs

Sur cette page, vous pouvez visualiser une carte sur laquelle sont disposés les différents producteurs de la région. L'icône de la petite maison marque votre emplacement géographique, alors que l'autre icône fait référence aux producteurs qui détiennent un compte dans notre application.

Conversation

Cette page montre la possibilité d'avoir une conversation avec d'autres utilisateurs de l'application. Comme vous pouvez le voir dans l'exemple, le client Dario est intéressé par un produit d'un producteur et décide de lui écrire un message pour demander si ce produit est disponible.

Liste de messagerie

Sur cette page vous pouvez voir un aperçu des conversations du producteur vinc-prod avec les autres utilisateurs de l'application.
